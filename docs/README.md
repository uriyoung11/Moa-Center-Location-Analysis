# Docs

프로젝트 설명과 발표/참고 자료를 모아둔 폴더입니다.

## Structure

```bash
docs/
├── presentation/
│   └── analysis_presentation.pptx
├── report/
│   └── analysis_summary.pdf
└── references/
```

## Contents

- `presentation/`: 발표용 슬라이드 원본
- `report/`: 대회 제출용 분석 요약 보고서
- `references/`: 분석 설계와 정책 배경 검토에 사용한 참고 자료

## Public Release Check

GitHub 공개 전 발표 자료와 보고서에서 아래 항목이 포함되어 있는지 확인하는 것을 권장합니다.

- 반출 제한 원본 데이터 표
- 개인정보 또는 식별 가능한 기관 내부 정보
- 로컬 절대경로
- 공개 권한이 불명확한 이미지나 지도 캡처
