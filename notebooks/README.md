# Notebooks

노트북은 작성자 기준이 아니라 분석 과정 기준으로 정리했습니다.

## 01_hvi_construction

주거취약지역을 정의하기 위한 HVI 산출 과정입니다.

- B068 및 건축물대장 기반 행정동 매핑
- 저가 주거, 노후 주택, 연립·다세대 집적도 산출
- HVI 점수/등급 산출
- HVI 지도 및 등급 시각화

## 02_policy_demand

정책 수요를 설명하는 하위 지표 산출 과정입니다.

- 노인
- 재난
- 치안
- 인프라
- 통신/생활 데이터

## 03_supply_accessibility

기존 모아센터 위치와 행정동 단위 접근성을 확인하는 과정입니다.

## 04_index_integration

HVI, 수요, 공급 공백 지표를 통합해 최종 우선순위 지표를 구성하는 과정입니다.

## 05_validation_visualization

최종 지표 검토, 결과 해석, 시각화 검증 과정입니다.
